Rei das Apostas

A simple webpage that scrapes the web and finds the best running offers for registering for sport betting websites. E.g. R$200 in bonus for signing up at Betano.

The pros

  • Great way to learn scraping, and setting a page with the scraped info.
  • Super easy to monetize once there's traffic.
  • Super valuable for the user once it's working.

The cons

  • A lot of technical SEO work to be done.
  • Not necessarily an easy scrape, as I might need to go into YT etc.

Priority Level: Medium

Ease to make: 2/5

A hard scrape job, but only a scrape job.

Ease to grow: 2/5

Tough but guaranteed via SEO. There might be guerrilla tactics.

Ease to monetize: 5/5

Monetization is guaranteed and high.

Fun & Heart: 3/5
